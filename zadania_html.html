Nagłowki do strony WWW.
  1. < h1> < /h1> Bedzie największym nagłówkiem
  2. od < h2> < /h2> do < h5> < /h5> Bedą coraz mniejsze
  3. < h6> < /h6> Będzie najmniejszym nagłówkiem
Podstawowe możliwości funkcji "font"
  1. < font size="oznacza wielkość czcionki">
  2. < font color="oznacza kolor czcionki>
  3. < font face="oznacza rodzaj czcionki">
Najprostsze rodzaje kolorów do użycia w funkcji "font"
  1. Nazwą: red - czerwony, green - zielony, blue - niebieski
  2. RGB: rgb(255,0,0) - czerwony, rgb(0,255,0) - zielony, rgb(0,0,255) - niebieski
  3. Hexadecymalny, #RRGGBB: #FF0000 - czerwony, #00FF00 - zielony, #0000FF - niebieski
